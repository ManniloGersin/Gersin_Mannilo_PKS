# Personal Digital Portfolio

## Description
This project is a personal digital portfolio website developed to showcase my skills, projects, certifications, and experiences as a Bachelor of Science in Information Technology (BSIT) student. The portfolio serves as an online professional profile that highlights my technical background, academic achievements, and career interests in areas such as mobile development, cybersecurity, networking, and user experience design.

The portfolio was built using modern web technologies including React, Next.js, and Tailwind CSS. It is deployed online using Vercel and connected to GitHub for version control and project management.
